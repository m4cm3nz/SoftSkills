# SoftSkills — The Superpower Soft Skill

> 🔗 **Ao vivo:** https://m4cm3nz.github.io/SoftSkills/

Uma landing page sobre **adaptabilidade** — a "superpower soft skill".

## A história

Esta foi a minha primeira página em React — meu "hello world" no framework.

Ela nasceu em novembro de 2020, como um dos desafios da Expansion Week do bootcamp
**GoStack** (Rocketseat): implementar uma landing page descrevendo a soft skill com a
qual você mais se identifica. Escolhi adaptabilidade, e a página defende a tese com uma
citação (falsamente) atribuída a Darwin, um verbete de dicionário e três super-heróis
mutantes.

Anos depois, a página fez jus ao próprio tema e se adaptou:

- **2020** — nasce com React 17, Webpack 5 e Babel, no calor do bootcamp.
- **2026** — migra para **Vite + React 18**, ganha layout responsivo, markup semântico,
  texto revisado (links mortos trocados por referências vivas) e deploy automático no
  GitHub Pages.

Se não é a mais forte nem a mais inteligente das landing pages, que seja a mais
adaptável. 🦎

## Stack

- [React 18](https://react.dev/)
- [Vite 6](https://vitejs.dev/) (dev server + build)
- CSS puro — sem framework de UI, de propósito

## Rodando localmente

```bash
npm install      # instala as dependências
npm run dev      # sobe o dev server (http://localhost:5173)
npm run build    # gera o build de produção em dist/
npm run preview  # serve o build de produção localmente
```
